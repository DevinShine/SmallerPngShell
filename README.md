# SmallerPngShell

![demo.gif-104.6kB][1]

###使用说明

将需要压缩的图片放到src文件夹下，然后在终端运行
```
python start.py
```
然后就会将压缩的图片输出至out文件夹。

相关的压缩参数可以查阅[pngquant](https://github.com/pornel/pngquant)的文档，目前脚本写死了一个我常用的参数，实际用于Android/iOS项目中并无太大问题。
